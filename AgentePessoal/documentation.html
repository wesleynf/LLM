🤖 Agente de Habilidades ReAct (Versão Python)

Este projeto representa a migração e evolução do sistema de orquestração de agentes de TypeScript para Python, focado em alta disponibilidade, execução stateless e multi-modalidade.

🏗️ Arquitetura do Sistema

O sistema utiliza o padrão ReAct (Reasoning and Acting) para permitir que o LLM "pense" antes de agir, utilizando ferramentas dinâmicas para resolver problemas complexos em tempo real.

Fluxo de Execução

  1. Telegram Input: Recebe texto, áudio ou PDF.
  2. Controller: Prefetch de metadados e roteamento de intenção.
  3. Agent Loop: Ciclo de Pensamento -> Ação -> Observação.
  4. Failover Provider: Tenta Gemini (Google); se falhar, escala para Groq (Llama-3/Qwen).
  5. Output Handler: Formata a resposta final (Markdown, Voz ou Documento).

📂 Estrutura de Módulos

  • Core agent_controller.py: Gerencia o estado e o ciclo de vida da requisição.
  • Core agent_loop.py: Implementa o motor ReAct com parsing de regex para chamadas de ferramentas.
  • Tools tool_registry.py: Registro central de ferramentas (Busca Web, Leitura de URL, Calendário).
  • Providers provider_factory.py: Sistema de failover entre provedores de LLM.

🛠️ Habilidades (Skills) Dinâmicas

As habilidades são carregadas dinamicamente de arquivos SKILL.md. Cada skill define seu próprio sistema de "regras de ouro" e instruções de personalidade.

🚗 Avaliador Veicular Saga

Objetivo: Avaliação técnica de veículos (usado por novo) com cruzamento de dados FIPE e Mercado Real.

  • Gatilho: "Quanto vale meu carro?", "HB20 2022".
  • Ferramentas: search_web, read_url.

🧠 Psicóloga (Najara)

Objetivo: Assistência psicológica direta e provocativa baseada em teorias comportamentais e freudianas.

  • Estilo: Direto, sem rodeios, focado em quebra de padrões.

🚀 Como Iniciar

# 1. Instalar dependências
pip install -r requirements.txt

# 2. Configurar .env (Copiar de .env.example)
# 3. Rodar o bot
python main.py
